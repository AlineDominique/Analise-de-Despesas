# 📊 Análise de Despesas da Prefeitura de São José dos Campos

Projeto de análise exploratória de dados (EDA) desenvolvido para examinar os gastos da Prefeitura de SJC durante o período de **janeiro de 2017 a maio de 2018**.

---

## 📋 Contexto
Este estudo foi realizado como parte da disciplina de **Inteligência Artificial** na **FATEC São José dos Campos**, sob a orientação do Prof. Fernando Massanori. A análise utiliza dados extraídos do Portal da Transparência municipal para identificar padrões de investimento e estágios das despesas públicas.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Principais Bibliotecas:** * `Pandas`: Para limpeza, agrupamento e manipulação dos dados.
    * `Matplotlib`: Para geração de gráficos e visualização de dados.
* **Ferramenta:** Jupyter Notebook.

## 📂 Estrutura da Análise
O projeto segue o seguinte fluxo de processamento:
1. **Organização de Dados:** Tratamento de tipos de dados e estruturação do Dataframe.
2. **Análise de Recursos:** Visão geral da origem e destino dos recursos.
3. **Estágios das Despesas:** Comparativo semestral entre despesas:
   - **Empenhadas** (Comprometidas)
   - **Liquidadas** (Serviço verificado)
   - **Pagas** (Saída efetiva de caixa)
4. **Análise por Secretaria:** Identificação dos maiores centros de custo da prefeitura.
5. **Natureza das Despesas:** Classificação detalhada do tipo de gasto.

## 📈 Visualizações
A análise inclui gráficos horizontais detalhados que permitem visualizar rapidamente quais Secretarias e Subfunções receberam os maiores volumes de pagamento, facilitando a interpretação de dados complexos.
